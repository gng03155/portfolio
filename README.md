# Portfolio 

## Info

>Vue + Typescript 기반 프로젝트입니다. </br>
>Firebase의 hosting 서비스를 이용해서 배포했습니다.</br>
>개인 프로젝트 소개 , 간단한 개인정보 , Github,Notion 링크 탑재 , 이력서 열람 가능 </br>

## Install

```
//root dir
npm install
npm run serve
```
